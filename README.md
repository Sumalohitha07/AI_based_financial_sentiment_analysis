# 📈 AI-Powered Financial Sentiment Analysis

This is a Streamlit app that predicts sentiment (Bullish, Bearish, Neutral) from financial news headlines using NLP and machine learning.

## 🔧 Technologies Used
- Python
- Scikit-learn
- Streamlit
- Joblib
- TF-IDF Vectorizer
